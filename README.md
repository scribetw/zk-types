(Archived) Type definitions for ZK
=======================

Since ZK 9.6.0, the definition is embedded so this repo is no longer needed.

## Usage

1. modify `package.json` to add dependencies.
```js
"dependencies": {
    "zk-types": "^9.0.0"
}
```

2. modify `tsconfig.json`
```js
"compilerOptions": {
    "types": ["zk-types"]
}
```

3. done
